Copy every file in this folder into:
C:\xampp\htdocs\LoungeRoyale

Choose Replace for all files.

Important:
Do not copy only foot-services.html. Replace all six service booking pages:
- hand-services.html
- foot-services.html
- lash-services.html
- wax-services.html
- kiddie-services.html
- royale-deluxe-package.html

Also copy these helper files into the same LoungeRoyale root folder:
- booking_availability.js
- check_availability.php
- confirm_booking.php
- style.css

Database:
If you have not done it yet, run seed_staff.sql in phpMyAdmin inside the lounge_royale database.

Result:
Every service page will show the employee names and disable booked times for the selected employee only.
